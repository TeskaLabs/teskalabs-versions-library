# LogMan.io v25.47 Release Notes

## Breaking Changes

## Features

### LMio Collector

- Added source for Cisco Meraki.
- Support for multiple ODBC inputs and queries specified from WebUI.
- Increase the default buffer size limit to 500MB.
- Refactoring of WEC/WEF input.
- Support for preferred IP address in CommLink protocol.

## Upgrade Procedure v25.30 â†’ v25.47

### Before the upgrade

- Roles `*/lmio-admin` and `*/lmio-plus-admin` have been obsoleted. Users with these roles must be assigned role `*/admin` instead.

### Upgrade

1. Change the version of LogMan.io in your `/Site/model.yaml` file to `v25.47`:

    ```yaml
    applications:
      - name: "ASAB Maestro"
        version: v25.47
      - name: "LogMan.io"
        version: v25.47
    ```

    Apply the changes.

2. Wait until all services are pulled and started.

3. Apply the changes again. This ensures that the latest versions of libraries used by LogMan.io services are pulled.

4. Wait until all services are started again.

### After the upgrade

1. Add the **LMIO Chart** service definition to your `/Site/model.yaml` file:

    ```yaml
    services:
        lmio-chart:
			- node01
			- node02
			- node03
    ```

    Apply the changes.
